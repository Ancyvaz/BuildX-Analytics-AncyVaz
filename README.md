# Campus Placement Analytics Workspace

## 📋 Project Overview
This repository contains my end-to-end data analytics workflow mapping the academic and professional factors driving successful college career placements.

### 🎯 Dataset Selected
* **Theme:** Education (College Placement Records)
* **Why I Selected It:** Understanding what factors actually drive hiring decisions and starting salaries is highly relevant to my own career journey, making it the perfect real-world scenario to analyze.

### 🛠️ Tools Utilized
* **Python (Pandas / Google Colab):** Data structural transformation and feature engineering.
* **SQL (SQLite Engine):** Targeted multi-variable business segmentation.
* **Power BI Desktop:** Multi-page canvas build, DAX measures, and dashboard architecture.

### ❓ Business Questions Answered
1. Which specific student records clear the threshold for top-tier financial earners (packages $\ge$ ₹3,50,000)?
2. What are the total student enrollment counts and average performance metrics for each undergraduate stream?
3. What is the explicit financial premium associated with entering placement cycles with prior work experience?

### 📈 Core Highlight Insight
Data tracking reveals that while high academic scores clear primary corporate gateway cutoffs, **Prior Work Experience** drives maximum salary lift. Placed candidates with a background history of professional work experience command significantly higher starting compensation tiers across all streams.
